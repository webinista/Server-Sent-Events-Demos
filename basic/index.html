Server-sent event demos

What time is it?

Requirements: This demo uses server-sent events and classList (also known as DOMTokenList). If it doesn't work in your browser, upgrade or switch. The latest versions of Opera, Chrome/Chromium, Safari, and Firefox support these features.

The box below will update periodically with the server’s current time. The default interval is set by the user agent, and ranges from about 3-6 seconds, depending on the browser.
