What time is it?

Requirements: This demo uses server-sent events. If it doesn't work in your browser, upgrade or switch. Current browser support.

The box below will update periodically with the server’s current time (added to the top of the stack). The default interval is set by the user agent, usually between 3 and 6 seconds.
