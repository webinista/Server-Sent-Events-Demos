What kind of number is it?

Requirements: This demo uses server-sent events. If it doesn't work in your browser, upgrade or switch. Current browser support.

When an event is dispatched from the server, we can treat it as a DOM event. That means we can set an event listener and define an event handler. In this case, we are sending even and odd events from the server. In our script, we are listening for odd and even events. Even events will also report whether the number is a perfect square number.

The server is sending data once every two seconds.
