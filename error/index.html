Server-sent event demos

What time is it?

Requirements: This demo uses server-sent events. If it doesn't work in your browser, upgrade or switch. Current browser support.

The box below will update once per second (with a server-set interval) with the server’s current time for a maximum of 10 times.

That limit is set within the script running at the URI we are polling. When the limit is reached — when the server stops sending data — it will trigger an error. When an error event is fired, we will alert the user (that's you) and our script will close the connection.

Because this is considered a fatal error, the client will not try to reconnect.
